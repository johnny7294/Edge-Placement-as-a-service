Run the niginxinstall.sh script
give the server name as you want but do not include '.com' at the end

example :  if you want to configure www.sparta.com , then give sparta as the server name
give the ip address of the interface on which you want the server

The script nginxinstall.sh will install the nginx with yum 
The script siteconf.sh will create the site you want and adds the necessary files
